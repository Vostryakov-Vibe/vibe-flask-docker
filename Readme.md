# 🐳 VPe02: Vostryakov-Vibe Docker Session (v1.2)

Проект по контейнеризации микросервиса на Python/Flask. Реализован в связке **Windows + WSL 2 + Docker Desktop**. 

---

## 📋 О проекте
Этот репозиторий — результат настройки изолированной среды и развертывания REST API. Проект демонстрирует навыки проектирования эндпоинтов, работы с Docker-образами и настройки сетевого взаимодействия между контейнером и хостом.

### Технологический стэк:
*   **Окружение:** Windows 10/11, WSL 2 (Ubuntu/Debian)
*   **Оркестрация:** Docker Engine ✅
*   **Язык:** Python 3.9-slim
*   **Фреймворк:** Flask

---

## 🚀 Функционал (Endpoints)

Приложение поддерживает три маршрута для проверки работоспособности и логики:
1.  `GET /` — Проверка статуса (Healthcheck).
2.  `GET /info` — Информация о разработчике и окружении.
3.  `GET /calc/<a>/<b>` — Динамический расчет суммы чисел через URL-параметры.
