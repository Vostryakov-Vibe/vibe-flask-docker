# 🐳 VPe02: Vostryakov-Vibe Docker 

Реализация микросервиса на Python/Flask, упакованного в Docker-контейнер. Проект выполнен в связке **Windows + WSL 2 + Docker Desktop**.

---

## 📋 О проекте
Данный репозиторий демонстрирует навыки контейнеризации веб-приложений, настройки изолированных сред (venv) и проектирования REST API с динамической маршрутизацией.

### Технологический стэк:
*   **Окружение:** Windows 10/11, WSL 2 (Ubuntu/Debian)
*   **Оркестрация:** Docker Engine ✅
*   **Язык:** Python 3.9-slim (оптимизированный Linux-образ)
*   **Фреймворк:** Flask 3.0
