# 🐳 Контейнеризация Flask API: Vibe Check & Calc

Проект по развертыванию масштабируемого Flask-приложения в Docker. Демонстрирует навыки проектирования REST API с динамическими параметрами и контейнеризацию в среде **Windows + WSL 2**.

## 🛠 Реализованные эндпоинты

*   `GET /` — Проверка статуса контейнера (**vibe_check_passed**).
*   `GET /info` — Метаданные разработчика и среды окружения.
*   `GET /calc/<a>/<b>` — Динамический расчет суммы (с обработкой ошибок ввода).
